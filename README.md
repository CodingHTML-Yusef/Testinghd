# Polaris
The new unblocker.

### Hosting Localy
Make sure you have [git](https://git-scm.com/) and [nodejs](https://nodejs.org/) installed
````bash
git clone https://github.com/polarisdevs/polaris-V1.git
````
Access the directory:
````bash
cd polaris-V1
````
Start the server:
````bash
npm install
npm start
````
